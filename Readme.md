**Kotlin-проект решения задачи моделирования теплового процесса пластины золота**
- последовательное решение
- параллельное решение (Executors, ListOf<Future<*>)
- возможность настройки входных параметров
- индикация теплового процесса в консоли (псевдографика)


Консольное приложение для численного решения двумерного уравнения теплопроводности (явная разностная схема) с поддержкой последовательного и параллельного расчёта.

Для **запуска** приложения необходимо:
- Установленная Java Runtime Environment (JRE 17+).
- Windows Terminal или любой иной терминал с поддержкой ANSI escape-последовательности (24-битный цвет, TrueColor).
